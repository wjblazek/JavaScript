This page demonstrates the bubbling of alerts from a nested element to it's ancestor elements.



The alert in this 1st example does not bubble up.
Clicking on the inner most element alerts on it but the alert does not bubble up because
there are no alerts in the parent elements.

This is the top level FORM with no alert onclicking and no bubbling.
This is a nested DIV with no alert onclicking and no bubbling.

The alert in this example bubbles up 1 level.
Clicking on the inner most element alerts on it, which bubbles up to it's parent element but no further.
Clicking on the middle element also alerts but does NOT bubble up to the top element.

This is the top level FORM with no alert onclicking.
This is a nested DIV. Click here to alert but it does not bubble up.

The alert in this example bubbles up to the top level.
Clicking on the inner most element alerts on it, which bubbles up to it's parent element and then to the top element.
Clicking on the middle element also alerts and bubbles up to the top element.
Clicking on the top element alerts on that element only.
